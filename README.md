JS Coding Style
------------
* Tabs indentation. Tab size: 4 spaces.
* Put opening brackets on new line. 
```javascript
if(smt)
{
	...
}
else
{
	...
}

function foo() { bar(); } // ACCEPTABLE
```
* Use camelCase.
```javascript
function myFunction() { } // GOOD
function MyFunction() { } // BAD
function my_function() { } // BAD
```
